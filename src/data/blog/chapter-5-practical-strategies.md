前四章解决了“认知”和“底盘”问题，第五章开始解决最痛的现实问题：

**到底怎么买？什么时候卖？执行时怎么不被情绪带跑？**

Nana 以前每次下单都像在考试：怕买贵、怕踏空、怕买完就跌。她后来发现，真正让人轻松的不是“预测更准”，而是“有流程可执行”。

## 1）买入策略：用定投把择时焦虑变成纪律

### 1.1 定投（DCA）为什么对普通人有效

定投的核心不是“收益最高”，而是：

1. 降低一次性买在高点的概率；
2. 用时间平滑情绪波动；
3. 让投资动作自动化，减少主观干扰。

尤其是工资型现金流人群，定投天然契合“每月有新增资金”的节奏。

### 1.2 三种常见定投节奏

- 固定金额定投：每月固定金额，执行最简单；
- 估值分层定投：估值高少买，估值低多买；
- 回撤增强定投：当指数回撤达到阈值时加一笔。

对新手最友好的顺序是：

**先固定金额跑满 6~12 个月，再尝试分层增强。**

### 1.3 一个可落地模板

例如：每月发薪后 T+1 自动转入投资账户，并在固定日期买入宽基 ETF。

- 基础定投：每月 3000 元；
- 回撤增强：指数较近高点回撤 >10% 时，额外加投 1000 元；
- 现金上限：若现金仓位 < 3 个月可投资额，则暂停增强。

### 1.4 买入前“五问法”

每次下单前，用 30 秒问自己：

1. 这是按计划买，还是因为情绪买？
2. 这笔钱未来 3~5 年是否不用？
3. 买入后若下跌 15%，是否还能按计划持有？
4. 当前仓位是否超过上限？
5. 若现在不能解释买入逻辑，是否应该先暂停？

只要有两项回答不确定，就推迟 24 小时再决定。

## 2）持有策略：让“在场时间”成为优势

### 2.1 为什么长期持有通常优于频繁换手

频繁交易会持续消耗你的收益：

- 手续费与滑点；
- 错过反弹窗口；
- 情绪决策导致高买低卖。

长期持有的本质，是让企业盈利增长和复利时间共同工作。

### 2.2 持有不等于“什么都不做”

长期策略依然需要管理动作：

1. 按季度复盘（不是按分钟盯盘）；
2. 检查是否偏离目标配置；
3. 评估基本面是否发生结构变化。

“长期”是低频、系统化管理，不是放任不管。

### 2.3 让持有更稳的三个习惯

- 习惯1：把“每日看收益”改成“每月看净投入与执行率”；
- 习惯2：用日历提醒复盘，减少临场判断；
- 习惯3：记录每次冲动交易前后的结果，形成反馈闭环。

## 3）卖出策略：卖出要靠规则，不靠感觉

卖出通常比买入更难，因为会触发贪婪和后悔。

### 3.1 三类常用卖出规则

1. **再平衡卖出**：某类资产占比超上限，卖出超出部分；
2. **目标兑现卖出**：达到预设收益区间，分批落袋；
3. **逻辑失效卖出**：当初买入理由被证伪时退出。

### 3.2 止损怎么设更实用

止损并不只适用于短线。中长期同样需要“风险上限”：

- 单笔资产最大亏损阈值（如 -15% 或 -20%）；
- 组合层面最大回撤阈值（如 -12%）；
- 触发后强制降风险仓位。

关键是提前写在计划里，而不是亏损时临时决定。

### 3.3 分批卖出模板（降低后悔）

如果你总担心“卖早了”，可以采用分批规则：

- 达到目标收益区间先卖出 20%~30%；
- 若继续上涨，再按固定步长逐步兑现；
- 若回撤到预设阈值，暂停追涨，回归组合目标。

这样既能锁定部分成果，也不至于完全踏空后续行情。

## 4）仓位管理：先活下来，再追收益

很多投资错误不是“方向错”，而是“仓位太重”。

### 4.1 仓位上限建议

- 单一高波动资产：不超过总资产的 10%~15%；
- 单一行业主题：不超过风险资产的 20%~30%；
- 单次新增仓位：避免一次打满，采用 2~4 次分批。

### 4.2 仓位调整三原则

1. 新资金优先补低配；
2. 触发上限再减仓；
3. 大波动时不临时翻倍加杠杆。

## 5）执行系统：把决策变成清单

Nana 最终把“策略”写成了每月固定流程：

### 每月第一个周末（例行）

- [ ] 更新收入、支出、储蓄率；
- [ ] 执行本月定投；
- [ ] 记录组合权重变化；
- [ ] 检查是否触发再平衡阈值；
- [ ] 写 3 行复盘（做对了什么/做错了什么/下月改进）。

### 大波动时（应急）

- [ ] 先看计划，不看群聊；
- [ ] 若未触发规则，不做临时性交易；
- [ ] 若触发阈值，按预设步骤执行，不加主观判断。

### 年度复盘（升级）

- [ ] 汇总全年净投入、净收益、最大回撤；
- [ ] 判断策略偏差来自“执行”还是“模型”；
- [ ] 仅调整一个关键参数进入下一年。

## 6）常见错误与对应修正

- 错误1：一次性满仓后失眠。  
  修正：改为分批买入 + 保留安全垫现金。

- 错误2：涨了舍不得卖，跌了不敢加。  
  修正：提前写分批止盈与再平衡规则。

- 错误3：每周换一个“新策略”。  
  修正：同一策略至少跟踪 2~4 个季度再评估。

- 错误4：把短期资金拿去做长期投资。  
  修正：按目标期限分桶，短钱短配，长钱长配。

- 错误5：拿结果归因能力，忽视运气。  
  修正：关注过程指标（执行率、纪律）而非单次盈亏。

## 7）实战演练：Nana 的 90 天执行计划

- 第 1~30 天：固定金额定投，建立记录表；
- 第 31~60 天：加入回撤增强规则；
- 第 61~90 天：第一次季度复盘并校准仓位。

目标不是“90 天赚多少”，而是形成可复制动作。

## 本章小结

第五章的重点不是“哪只资产最猛”，而是建立一套你能长期执行的买卖流程。

- 买入靠定投和分批，不赌单点时机；
- 持有靠复盘和再平衡，不靠盯盘；
- 卖出靠规则和阈值，不靠情绪；
- 仓位管理决定你能否穿越周期。

**把交易动作流程化，你的心理压力会显著下降，收益反而更稳定。**
